## 🚀 Deploy to GitHub Pages (5 minutes)

### Option 1: Direct Upload (Easiest)

1. Go to https://github.com/new
2. Create a repository named `guthealinglab` (or your preferred name)
3. On your computer, open the `guthealinglab` folder
4. Drag and drop all files and folders into the repository on GitHub
5. Go to Settings → Pages → Source → Deploy from branch `main` → `/ (root)`
6. Your site will be live at `https://yourusername.github.io/guthealinglab/`

### Option 2: Using Git (if you have Git installed)

```bash
cd guthealinglab
git init
git add .
git commit -m "Initial commit: Gut Healing Lab blog"
git remote add origin https://github.com/yourusername/guthealinglab.git
git push -u origin main
```

Then enable GitHub Pages in repository Settings → Pages.

### Custom Domain (Optional)

1. Buy a domain (e.g., guthealinglab.com) from Namecheap, Google Domains, etc.
2. In GitHub Pages settings, enter your custom domain
3. Add a CNAME record at your domain provider pointing to `yourusername.github.io`
4. Update the `sitemap.xml` and `robots.txt` URLs to use your custom domain
